# LazyStack project page

Project website for **LazyStack: Unifying Stacking and Cascading for Efficient Ensemble Inference** (ICML 2026).

Built with React + Vite. Custom interactive SVG components (teaser, animated cascade demo, method pipeline, trajectory landscape).

## Develop

```bash
npm install
npm run dev      # local dev server
npm run build    # production build to dist/
```

Deployed to GitHub Pages at https://ashwincolaco.github.io/lazystack/ (built `dist/` published to the `gh-pages` branch).
